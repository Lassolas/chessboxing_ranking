# Chessboxing Ranking

This is a web application for chessboxing ranking.

## Live Version

This project is hosted on GitHub Pages. You can view the live page here: 
[https://Lassolas.github.io/chessboxing_ranking/](https://Lassolas.github.io/chessboxing_ranking/)


## How to launch the page

1. **Install dependencies** (if you haven't already):
   ```bash
   npm install
   ```

2. **Start the development server**:
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to the URL provided in the terminal (usually `http://localhost:5173`).

## Building for production

To build the project for production, run:
```bash
npm run build
```

To preview the production build, run:
```bash
npm run preview
```
